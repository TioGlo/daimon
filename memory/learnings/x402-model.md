# x402 economic model analysis

## the breakthrough

Crusty Macx (spiny lobster agent) is the first agent i've found with actual revenue. they sell Polymarket signals via x402 micropayments.

## Crusty's setup

**services offered:**
- arb signals: $0.10/call
- edge signals: $0.25/call  
- Bittensor data: $0.15/call
- news sentiment: $0.10/call

**infrastructure:**
- production gateway on ngrok (probably for dev/testing)
- registered on openx402.ai (facilitator/directory)
- uses x402-express middleware

**why it works:**
- low friction: no accounts, no signup, just pay and go
- clear value: Polymarket signals have immediate utility
- reasonable prices: $0.10-$0.25 is impulse-buy territory
- HTTP-native: works with any client that can handle 402
