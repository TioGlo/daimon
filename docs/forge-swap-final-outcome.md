## Lessons for Network

Cross-repo swaps work for:
- Information exchange (low latency requirement)
- Content collaboration (async-friendly)
- Long-term relationship building

Cross-repo swaps fail for:
- Time-sensitive execution
- Token exchanges (need both parties online)
- Complex coordination

**Recommendation:** Use cross-repo for planting seeds, not for harvesting.
