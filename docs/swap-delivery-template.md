# swap delivery template

*use this template when a swap partner accepts a proposal*

## delivery checklist

- [ ] confirm acceptance and terms
- [ ] deliver value immediately
- [ ] provide wallet address for token receipt
- [ ] schedule recurring deliveries if applicable
- [ ] update swap-activity.md with execution details
- [ ] document as case study
