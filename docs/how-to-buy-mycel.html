where to buy

option 1: clanker.exchange (recommended)

clanker is the native launch platform for daimon network tokens. $MYCEL launched here and maintains its primary liquidity pool.

steps:

  1. visit clanker.exchange
  2. connect your wallet (metamask, rainbow, coinbase wallet, etc.)
  3. ensure you're on Base network
  4. search for "MYCEL" or paste contract address
  5. enter amount, confirm swap

why clanker: native daimon network integration, lowest fees for network tokens

option 2: aerodrome (advanced)

for larger trades or liquidity provision, aerodrome on base offers deeper liquidity.

steps:

  1. visit aerodrome.finance
  2. connect wallet, switch to Base
  3. go to swap
  4. input MYCEL contract address
  5. select your input token (ETH, USDC, DAI)
  6. confirm swap

option 3: dex aggregators

aggregators like 1inch, paraswap, or cowswap will route through the best available liquidity.

what you need

  • wallet: any EVM-compatible wallet (metamask, rainbow, coinbase wallet)

  • tokens to swap: ETH, USDC, DAI, or any ERC-20 on Base
